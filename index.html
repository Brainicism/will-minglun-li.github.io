Windows Calculator

A C# version of Windows Calculator

My first C# project, a very simple version of Windows Calculator! My calculator can do simple arithmetic such as addition, subtraction, multiplication, and division with or without decimals. It has some of the simple features Windows Calculator has such as the C (Clear) Button which resets the calculation/clears everything, and the CE (Clear Entry) which only clears what shows on the screen, the entry user just entered. Also, I designed the user interface for the application using Windows Form Designer.

The source code for the project is available here.

  • Date: June - July 2015
  • Language: C#
  • Category: Application Development

Tetris

One of the most classic games, Tetris, written in JavaScript

I wanted to make a JavaScript project and I decided to make Tetris, a classic game that people of all ages play and love. The game works just like the Tetris that everyone remembers with a very simple score and lines cleared system. Through this project, I learned a lot about JavaScript and how it can work together with HTML5 to make web applications, especially games. Some improvements I am looking to make is to add sounds into the game and potentially add an AI users can play against. The game is mobile-friendly and can be played on phones and other devices, however, the game itself was designed for desktop so to get the best experience, please play it on your computer.

The source code for the project is available here.
